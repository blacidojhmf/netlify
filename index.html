COE MIDIS – Datos de ejemplo

Este sitio está pensado para simular una fuente JSON para la página de monitoreo. La URL que debes pegar en la app es:

https://TU-DOMINIO/data.json

Contenido del data.json (formato esperado):

[
  { "region": "Piura",  "titular": "Thais Mejía", "suplente": "Paul Zamora" },
  { "region": "Cusco",  "titular": "María Huamán", "suplente": "Jorge Paredes" },
  { "region": "Loreto", "titular": "Ana Rivas",   "suplente": "Luis Carranza" }
]

Sube esta carpeta a Netlify Drop o publícala en GitHub Pages. Luego copia la URL pública de data.json y pégala en la app.

Si necesitas cambiar o ampliar los datos, edita el archivo data.json.
